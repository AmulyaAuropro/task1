LOWER AND UPPER CASE FROM LIST:
x=input("Enter the string:- ")
def char(x):
  u=0
  l=0
  for i in x:
      if i>='a' and i<='z':
       l+=1

      if i >='A' and i<='Z':
       u+=1

  print("LowerCase letter in the String",l)
  print("UpperCase letter in the String",u)
char(x)

Output:
Enter the string:- I'm Python User
LowerCase letter in the String 9
UpperCase letter in the String 3
